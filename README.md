# UX Editor

UX Editor is a Symfony bundle integrating the [Editor.js][1] library in Symfony applications.

[1]: https://editorjs.io

## Requirements 
- PHP 8.2 or higher
- StimulusBundle
- Composer

## Installation

Install the library via Composer:

```console
composer require pentiminax/ux-editor
```
